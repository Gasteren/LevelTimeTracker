# Level Time Tracker

Level Time Tracker is a World of Warcraft addon that tracks how long it takes to level, per level and per character.

It provides accurate, persistent timing, live XP/hour calculations, and a clean in-game UI with compact and full display modes.

---

## Features

- Tracks time spent on each level
- Accurate XP-based timing (starts only when XP is gained)
- Live XP/hour calculation (session-based)
- Progress bar with partial XP support
- Persistent timers across reloads and relogs
- Handles dungeon XP bursts and multi-level skips
- Compact and full UI modes
- Chat message on every level-up
- Expansion-proof (dynamic max level detection)

---

## Installation

1. Download the addon from CurseForge

  or Manually 
  
3. Extract into: World of Warcraft/retail/Interface/AddOns/LevelTimeTracker
4. Enable the addon in-game

---

## Usage

Open the UI: /ltt

- Compact mode (C) is ideal for keeping the tracker open while leveling
- Full mode (F) shows detailed per-level breakdowns

---

## How It Works

- Timing begins only when the player gains XP
- Timers persist across reloads and relogs
- XP/hour resets per session to stay accurate
- Level-up timing is recorded even for large XP jumps
